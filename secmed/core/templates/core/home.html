Welcome to Secure Med

Secure Med is a shared user interface between Network Administrators and Biomed that allows these teams to work together to lock down the security of medical devices on the network, while at the same time providing speed and ease required for Biomed to deliver quality Healthcare. Secure Med automates the provisioning, inventory and network security using Cisco DNA Center, Cisco Identity Services Engine on a Medical Grade Network. 

Biomed Interface

Secure Med's Biomed interface allows the Biomed team to:
- manage medical device inventory
- discover and add new medical devices to the network
- secure medical devices via predefined IT profiles
- request new IT security profiles for new medical device models

Network Administrator Interface

Secure Med's Net Admin interface allows the IT team to:
- Manage service requests from the Biomed for new medical devices
- Get a snapshot view of the medical devices on the network
- Write access profiles for medical devices based on manufacture and model
